Klone Helm Charts Repository

📦 Add Repository

Add this Helm repository to your local Helm installation:

helm repo add klone https://raghavendiran-2002.github.io/klone/helm-charts
helm repo update

🚀 Install Klone Operator

helm install klone-operator klone/klone-operator \
  --version 1.0.54 \
  --namespace klone \
  --create-namespace
